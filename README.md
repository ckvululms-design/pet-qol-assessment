# 凝視犬貓生活品質評估

公開使用的犬貓生活品質評估工具，整合四份量表：

- 狗心臟病健康相關生活品質 (FETCH) 評估
- 狗健康相關生活品質 (HRQL) 評估
- 貓咪高齡與安寧照護 (QOL) 評估
- 癌症動物健康相關生活品質 (HRQoL) 評估

## 網址架構建議

本專案採用「一個網站入口，四個獨立路由」：

- `/dog-fetch`
- `/dog-hrql`
- `/cat-qol`
- `/cancer-qol`

優點是使用者只要記一個網站，Vercel 也只需要維護一個專案；同時每份量表仍能單獨分享網址。缺點是四份量表共用同一套版面，若未來要做成完全不同品牌或不同單位管理，拆成四個網站會更獨立。

四個完全不相關網址的優點是每份量表可以各自調整視覺、權限、SEO 與版本；缺點是維護成本高，使用者也比較容易找不到其他量表。

## 功能

- 手機優先的大字體與大分數按鈕
- 自動計算總分、生活品質指標、完成度與分類小計
- 支援反向題提示
- 支援備註、寵物姓名或代號、填寫日期
- 支援列印 / 儲存 PDF
- 支援複製摘要與複製目前量表連結
- 不需要後端，資料只保存在使用者目前裝置的瀏覽器

## Vercel 部署

這是純靜態網站，不需要 build command。

1. 將此 repo 推到 GitHub。
2. 在 Vercel 匯入 GitHub repo。
3. Framework Preset 選 `Other` 或保持自動偵測。
4. Build Command 留空。
5. Output Directory 留空或使用專案根目錄。

`vercel.json` 已設定四個路由都回到 `index.html`，使用者可直接打開任一量表網址。
